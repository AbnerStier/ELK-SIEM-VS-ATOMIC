# 🛡️ Active Directory + SIEM Home Lab

Hello There! I'm Abner 👋  

This lab environment is inspired by the knowledge I gained through the **SIEM modules in the Hack The Box SOC Path**.  
It has been designed exclusively for **educational purposes**, serving as a hands-on platform to practice, experiment, and reinforce concepts related to **Security Information and Event Management (SIEM)** using the **ELK stack**.  

In addition to the defensive perspective, this lab also incorporates a **Red Team simulation** to emulate real-world attack scenarios.  
The goal is to provide a safe and controlled environment where both offensive and defensive techniques can be explored, helping to understand how threats are detected, analyzed, and mitigated within a SIEM solution.
